9.1 Pendahuluan

Item Response Theory (IRT) mengevaluasi soal melalui hubungan probabilistik antara kemampuan peserta (θ) dan peluang menjawab benar. Dibandingkan CTT, IRT memberikan:

Aspek CTT IRT
Parameter item Tergantung sampel Invariant (bebas sampel)
Parameter peserta Skor total mentah Kemampuan θ pada skala logit
Informasi pengukuran Satu nilai (α) Kurva — berbeda di tiap level θ
Model utama — 1PL, 2PL, 3PL

Pada contoh ini kita gunakan Model 2PL dengan dua parameter per soal:

  • a = daya beda (discrimination) — kemiringan kurva ICC
  • b = kesulitan (difficulty) — posisi kurva pada skala θ

\[P(X=1 \mid \theta, a, b) = \frac{1}{1 + e^{-a(\theta - b)}}\]

9.4 Item Characteristic Curves (ICC)

ICC menunjukkan hubungan antara kemampuan θ dan probabilitas menjawab benar.

Code 
theta_seq <- seq(-4, 4, by = 0.05)

# Buat data ICC
df_icc <- do.call(rbind, lapply(1:n_soal, function(i) {
  data.frame(
    theta = theta_seq,
    prob  = 1 / (1 + exp(-a_est[i] * (theta_seq - b_est[i]))),
    soal  = paste0("Soal_", sprintf("%02d", i)),
    a     = round(a_est[i], 2),
    b     = round(b_est[i], 2)
  )
}))

# Label legend per soal: "Soal_01  a=0.80  b=-1.50"
soal_ids <- paste0("Soal_", sprintf("%02d", 1:n_soal))

label_legend <- sapply(soal_ids, function(s) {
  r <- df_icc[df_icc$soal == s, ][1, ]
  sprintf("%-9s  a=%4.2f  b=%+5.2f", s, r$a, r$b)
})

df_icc$soal_f <- factor(df_icc$soal,
                         levels = soal_ids,
                         labels = label_legend)

# 15 warna kontras
pal_15 <- c(
  "#e74c3c", "#c0392b", "#e67e22", "#f39c12", "#d4ac0d",

9.7 Test Information Function (TIF)

TIF menunjukkan di rentang θ mana tes paling akurat mengukur.

Code 
# Hitung IIF per soal lalu jumlahkan
iif <- function(theta, a, b) {
  p <- 1 / (1 + exp(-a * (theta - b)))
  a^2 * p * (1 - p)
}

tif_vals <- rowSums(sapply(1:n_soal, function(i)
  iif(theta_seq, a_est[i], b_est[i])))
sem_vals <- 1 / sqrt(tif_vals)

df_tif <- data.frame(
  theta = theta_seq,
  TIF   = tif_vals,
  SEM   = pmin(sem_vals, 3)   # cap SEM agar tidak meledak di ujung
)

p_tif <- ggplot(df_tif, aes(x = theta, y = TIF)) +
  geom_area(fill = "#3498db", alpha = 0.3) +
  geom_line(color = "#2980b9", linewidth = 1.2) +
  geom_vline(xintercept = theta_seq[which.max(tif_vals)],
             linetype = "dashed", color = "#e74c3c") +
  labs(title = "Test Information Function",
       subtitle = sprintf("Informasi maksimum pada \u03b8 = %.2f",
                          theta_seq[which.max(tif_vals)]),
       x = "\u03b8", y = "Informasi I(\u03b8)") +
  theme_minimal(base_size = 12)

p_sem <- ggplot(df_tif, aes(x = theta, y = SEM)) +
  geom_area(fill = "#e74c3c", alpha = 0.25) +
  geom_line(color = "#c0392b", linewidth = 1.2) +
  geom_hline(yintercept = 0.45, linetype = "dashed", color = "gray50") +
  annotate("label", x = 3, y = 0.45,
           label = "SEM = 0.45", fill = "white",
           color = "gray40", size = 3) +
  labs(title = "Standard Error of Measurement",
       subtitle = "SEM rendah = pengukuran lebih presisi",
       x = "\u03b8", y = "SEM(\u03b8)") +
  theme_minimal(base_size = 12)

## Pendahuluan

**Item Response Theory (IRT)** mengevaluasi soal melalui hubungan probabilistik antara kemampuan peserta (θ) dan peluang menjawab benar. Dibandingkan CTT, IRT memberikan:

| Aspek | CTT | IRT |
|-------|-----|-----|
| Parameter item | Tergantung sampel | Invariant (bebas sampel) |
| Parameter peserta | Skor total mentah | Kemampuan θ pada skala logit |
| Informasi pengukuran | Satu nilai (α) | Kurva — berbeda di tiap level θ |
| Model utama | — | 1PL, 2PL, 3PL |

Pada contoh ini kita gunakan **Model 2PL** dengan dua parameter per soal:

- **a** = daya beda (*discrimination*) — kemiringan kurva ICC
- **b** = kesulitan (*difficulty*) — posisi kurva pada skala θ

$$P(X=1 \mid \theta, a, b) = \frac{1}{1 + e^{-a(\theta - b)}}$$

## Test Information Function (TIF)

TIF menunjukkan di rentang θ mana tes paling akurat mengukur.

```{r}
#| label: fig-tif
#| fig-cap: "Test Information Function & Standard Error of Measurement"
#| fig-height: 5

# Hitung IIF per soal lalu jumlahkan
iif <- function(theta, a, b) {
  p <- 1 / (1 + exp(-a * (theta - b)))
  a^2 * p * (1 - p)
}

tif_vals <- rowSums(sapply(1:n_soal, function(i)
  iif(theta_seq, a_est[i], b_est[i])))
sem_vals <- 1 / sqrt(tif_vals)

df_tif <- data.frame(
  theta = theta_seq,
  TIF   = tif_vals,
  SEM   = pmin(sem_vals, 3)   # cap SEM agar tidak meledak di ujung
)

p_tif <- ggplot(df_tif, aes(x = theta, y = TIF)) +
  geom_area(fill = "#3498db", alpha = 0.3) +
  geom_line(color = "#2980b9", linewidth = 1.2) +
  geom_vline(xintercept = theta_seq[which.max(tif_vals)],
             linetype = "dashed", color = "#e74c3c") +
  labs(title = "Test Information Function",
       subtitle = sprintf("Informasi maksimum pada \u03b8 = %.2f",
                          theta_seq[which.max(tif_vals)]),
       x = "\u03b8", y = "Informasi I(\u03b8)") +
  theme_minimal(base_size = 12)

p_sem <- ggplot(df_tif, aes(x = theta, y = SEM)) +
  geom_area(fill = "#e74c3c", alpha = 0.25) +
  geom_line(color = "#c0392b", linewidth = 1.2) +
  geom_hline(yintercept = 0.45, linetype = "dashed", color = "gray50") +
  annotate("label", x = 3, y = 0.45,
           label = "SEM = 0.45", fill = "white",
           color = "gray40", size = 3) +
  labs(title = "Standard Error of Measurement",
       subtitle = "SEM rendah = pengukuran lebih presisi",
       x = "\u03b8", y = "SEM(\u03b8)") +
  theme_minimal(base_size = 12)
```
